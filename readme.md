#Vim_oNote


Vim_oNote is a project which keeps...

1- A Vim cheat sheet.
2- A Vim oNote.


***Vim oNote project will not keep/deal with any GVIM command.

------------------------------------

* [Stackoverflow Vim Brainstorming Sessions](http://stackoverflow.com/questions/tagged/vim)
