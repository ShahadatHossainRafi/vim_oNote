#Vim_oNote

Vim_oNote is a project which keeps...

1- A Vim cheat sheet.
2- A Vim oNote.

***Vim oNote project will not keep/deal with any GVIM command.

Vim oNote project is created based on...

1- Stackoverflow answers.
2- http://www.openvim.com/
